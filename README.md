# AI Chatbot (MERN Stack)

An AI-powered chatbot web application built using the MERN stack (MongoDB, Express, React, Node.js) and OpenAI API integration.

## 🚀 Features

- AI-powered chat interaction using OpenAI API
- Responsive React frontend
- Node.js/Express backend API handling
- Secure environment variable management via `.env`
- Deployed frontend on Vercel
- Backend hosting setup (Render, Railway, or local)
